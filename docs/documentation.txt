General specifications:
Input: File
Output: Terminal
Variables naming: only letters
Each statement must begin with new line
Each element from a math expression must be separated by a space
Comments start with # and free lines are acceptable



Existing instructions:
<int/float> <varname> = <number>
<varname> = <math expression>
show <varname>

if <varname> = <number> then
endif

(math expression are made from operators +, -, /, * and variables/integers/float numbers)
(if statements can be included one in another)

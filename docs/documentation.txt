General specifications:
Input: File
Output: Terminal
Variables naming: only letters
Case sensitive
Each statement must begin with new line
Comments start with #



Existing instructions:
# Comments

<varname> <+/-/*/ / /**>= <math expression>
show <varname>

Example of a program:
int a=12
# Variable a - initialized
a /= 2 + 3
show a
